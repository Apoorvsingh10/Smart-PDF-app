```
Prepare PDFKitPrivate for Play Store release.

Requirements:
- Release build configuration
- App signing
- 16KB page alignment (Android 15)

Update CMakeLists.txt for release:

- Strip debug symbols
- Enable LTO (Link Time Optimization)

Update android/AndroidManifest.xml:
- Verify NO internet permissions
- Add proper app metadata
- Set android:extractNativeLibs="true"
- Target SDK 35
- Min SDK 24

Update android/build.gradle:
- Configure signing (placeholder for keystore)
- Enable R8/ProGuard
- Set 16KB page alignment:
  packagingOptions {
    jniLibs {
      useLegacyPackaging = true
    }
  }
- Generate AAB (Android App Bundle)

Create android/res/ assets:
- ic_launcher.png (all densities: mdpi, hdpi, xhdpi, xxhdpi, xxxhdpi)
- Suggest: Shield icon with "PDF" text, green accent
- feature_graphic.png (1024x500) for Play Store
- screenshots/ folder structure

Create STORE_LISTING.md with:
- App name: "PDFKit Private - 100% Offline PDF Tools"
- Short description (80 chars): "Compress, merge, convert PDFs offline. Your files never leave your device."
- Full description (4000 chars): Privacy-focused messaging, feature list, comparison to cloud apps
- Keywords: pdf offline, pdf compressor no internet, private pdf editor, pdf to word offline
- Category: Productivity
- Content rating: Everyone
- Privacy policy URL placeholder

Create README.md:
- Build instructions
- Feature list
- Architecture overview
- License (your choice)

Build commands:
```
